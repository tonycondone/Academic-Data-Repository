## Conversion Steps to Microsoft Word

### Step 1: Open the HTML File
1. Open `Final_Year_Project_Report_TTU_Complete.html` in your web browser
2. Press `Ctrl+A` to select all content
3. Press `Ctrl+C` to copy

### Step 2: Create Word Document
1. Open Microsoft Word
2. Create a new blank document
3. Go to **Layout** → **Margins** → **Custom Margins**
   - Top: 25mm (0.98 inches)
   - Left: 25mm (0.98 inches)
   - Bottom: 20mm (0.79 inches)
   - Right: 20mm (0.79 inches)

### Step 3: Paste and Format
1. Press `Ctrl+V` to paste the content
2. Select all text (`Ctrl+A`)
3. Set font to **Times New Roman**
4. Set line spacing to **1.5**

### Step 4: Apply Heading Styles
1. **Title Page Elements**: 
   - University name: 16pt, Bold, Center
   - Project title: 14pt, Bold, Center
   - Other elements: 12pt, Center

2. **Chapter Headings**: 
   - Format → Styles → Heading 1
   - Modify: 14pt, Bold, Center, Space Before: 30pt

3. **Section Headings (H2)**: 
   - Format → Styles → Heading 2
   - Modify: 12pt, Bold, Left

4. **Subsection Headings (H3)**: 
   - Format → Styles → Heading 3
   - Modify: 12pt, Bold, Left

### Step 5: Add Page Numbers
1. Insert → Page Number → Bottom of Page → Plain Number 2 (centered)
2. Different First Page (no number on title page)
3. Format Page Numbers:
   - Title page through Table of Contents: Roman numerals (i, ii, iii...)
   - Chapter 1 onwards: Arabic numerals (1, 2, 3...)

### Step 6: Insert Visual Elements

#### Entity Relationship Diagram (for Section 3.5.1)
Create using Word's SmartArt or Shapes:
1. Insert → SmartArt → Hierarchy → Organization Chart
2. Or use Insert → Shapes to create custom diagram

#### System Architecture Diagram (for Section 3.4.1)
1. Insert → Shapes → Rectangle
2. Create three stacked boxes for:
   - Presentation Tier
   - Application Tier
   - Data Tier

#### Sample Screenshots (for Chapter 4)
1. Take actual screenshots from your system
2. Insert → Pictures → From File
3. Add captions: References → Insert Caption

### Step 7: Format Tables
1. Ensure all tables have:
   - Borders on all cells
   - Header row with gray background
   - Caption above table
   - Consistent formatting

### Step 8: Update Table of Contents
1. Place cursor where TOC should be
2. References → Table of Contents → Custom
3. Show levels: 3
4. Use dots for tab leader

### Step 9: Final Checks
- [ ] All pages have correct margins
- [ ] Font is Times New Roman throughout
- [ ] Line spacing is 1.5
- [ ] Page numbers are correct
- [ ] All figures and tables have captions
- [ ] References are in APA format
- [ ] No spelling or grammar errors

## Creating the PDF Version

### From Word:
1. File → Save As
2. Choose PDF format
3. Options → Ensure "Document structure tags" is checked
4. Save

### Quality Check:
1. Open PDF and verify:
   - All formatting preserved
   - Page breaks correct
   - Images and diagrams clear
   - Text searchable
